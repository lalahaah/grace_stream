[Grace Stream] 홈 화면 감정 카테고리 기획 의도홈 화면 중앙에 배치된 **'감정 아이콘(위로, 감사, 평안, 용기)'**은 단순한 메뉴를 넘어, 사용자의 현재 영적 상태와 앱의 콘텐츠를 실시간으로 연결하는 핵심적인 UX 장치입니다.1. 사용자의 심리적 진입 장벽 완화 (Emotional UX)의도: 사용자가 성경 앱을 켜는 이유는 단순히 지식을 얻기 위함이 아니라, 현재의 답답함이나 기쁨을 신앙적으로 해소하고 싶기 때문인 경우가 많습니다.효과: "무엇을 읽을까?" 고민하게 만드는 대신, "지금 내 마음이 어떤가?"를 먼저 묻고 그에 맞는 '위로'나 '용기'의 메시지를 즉각 제공하여 심리적 만족감을 높입니다.2. AI 감성 태깅의 시각화 (AI-Driven Curation)의도: 기획안의 핵심 기술인 Gemini API를 통해 성경 구절과 CCM 가사를 감정 키워드로 자동 분류(Tagging)하고 있음을 사용자에게 보여주는 UI입니다.효과: 사용자가 '평안' 아이콘을 누르면, AI가 분석한 평안 관련 성경 구절과 그 분위기에 어울리는 워십 영상이 세트로 큐레이션되어 나타납니다.3. 플랫폼 비즈니스를 위한 데이터 확보 (Data-Informed Growth)의도: 사용자가 어떤 아이콘을 가장 많이 클릭하는지 추적하여 플랫폼의 콘텐츠 방향성을 결정합니다.효과: 예를 들어 '위로' 카테고리의 클릭률이 높다면, 해당 테마의 신규 CCM 아티스트를 메인에 더 노출하거나 관련 굿즈(예: 위로의 구절 카드 등)를 추천하는 비즈니스 확장이 가능합니다.4. 상황별 퀵 액세스 (Quick Access)의도: 바쁜 현대인이 짧은 시간 안에 핵심 묵상을 마칠 수 있도록 돕는 '숏컷(Shortcut)' 기능입니다.효과: 특정 주제를 검색할 필요 없이 아이콘 하나로 관련 성경 본문 + 해설 + 찬양 플레이어를 동시에 활성화하여 서비스 체류 시간과 사용성을 극대화합니다.💡 요약이 아이콘들은 사용자의 감정 → 성경의 지혜 → 찬양의 감동을 하나의 선으로 연결하는 '감성적 큐레이션 통로' 역할을 합니다.
