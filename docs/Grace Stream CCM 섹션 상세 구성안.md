# **\[Grace Stream\] CCM & 워십 섹션 상세 기획안**

사용자가 자신의 영적 상태에 맞는 찬양을 발견하고, 아티스트와 소통하며, 말씀과 연결된 찬양을 경험할 수 있도록 구성합니다.

## **1\. 섹션 상단: 오늘의 픽 (Today's Featured)**

사용자의 취향과 오늘의 성경 통독 목표를 결합한 큐레이션 영역입니다.

* **AI 맞춤형 추천:** "오늘 읽으신 창세기 1장과 어울리는 웅장한 창조 찬양"과 같은 개인화된 배너.  
* **이달의 워십 리더:** 새롭게 떠오르는 인디 CCM 아티스트나 워십팀 인터뷰 및 대표곡 소개 (플랫폼 비즈니스 확장 포인트).

## **2\. 감정/상황별 테마 (Mood & Context)**

홈 화면의 감정 아이콘과 연동되는 상세 탐색 영역입니다.

* **테마 구성:** \* 🕊️ **평안:** 잠자기 전 듣기 좋은 연주곡 또는 잔잔한 고백.  
  * 🔥 **뜨거운 찬양:** 집회 현장의 열기를 담은 파워풀한 워십.  
  * 🌿 **묵상:** 가사에 집중할 수 있는 어쿠스틱 찬양.  
  * 🦁 **용기:** 고난 중에 힘이 되는 선포형 찬양.

## **3\. 하이브리드 플레이어 구성 (Hybrid Player UX)**

음악과 영상을 자유롭게 오가는 사용자 경험을 제공합니다.

* **영상-오디오 전환 토글:** 유튜브 API를 활용하여, 영상으로 보고 싶을 때는 화면을 띄우고, 이동 중에는 오디오 모드로 전환 가능.  
* **성경 구절 팝업:** 찬양 가사 중에 인용된 성경 구절이 있을 경우, 클릭 시 해당 구절로 바로 이동하는 기능.  
* **찬양 필사/노트:** 찬양을 들으며 떠오르는 영감을 바로 기록하는 하단 텍스트 필드.

## **4\. 플랫폼 확장: 아티스트 허브 (Artist Hub)**

인디 아티스트들이 자신의 채널을 홍보하고 유저와 만나는 공간입니다.

* **공식 채널 연동:** 아티스트의 유튜브 공식 채널과 직접 연결하여 조회수와 수익이 아티스트에게 직접 돌아가도록 설계.  
* **후원 및 응원:** 아티스트에게 응원 메시지를 남기거나, 나중에 플랫폼이 커졌을 때 가상 굿즈를 선물하는 기능.

## **5\. UI 위젯 레이아웃 (Flutter Widget Structure)**

1. **Horizontal Scroll (수평 스크롤):** 상단 배너 및 인기 아티스트 목록.  
2. **Grid View (그리드):** 감정별/상황별 카테고리 (2x2 혹은 2x4 형태).  
3. **Vertical List (수직 리스트):** 최근 들은 찬양, 내가 저장한 플레이리스트.  
4. **Floating Player (플로팅 플레이어):** 어떤 화면에서도 재생 중인 곡 정보를 확인하고 제어하는 미니 바.

## **6\. 저작권 기반 운영 전략 (Copyright Strategy)**

* **YouTube Data API v3 활용:** 모든 곡 정보와 썸네일을 유튜브 API에서 가져와서 실시간 업데이트.  
* **플레이리스트 공유:** 사용자가 만든 플레이리스트를 친구에게 카카오톡 등으로 공유하여 플랫폼 유입 유도.
